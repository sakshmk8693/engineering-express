# Engineering Express ADMIN & User App

Engineering Express is a secure educational app designed for students to
access notes and learning materials safely.

# Features
- Secure login & device-based access
- Encrypted content delivery
- OTP-based authentication
- Notes download inside app only
- Anti-leak & security focused architecture

# Tech Stack
- Android (Java/Kotlin)
- Supabase (Auth & DB)
- Google Drive (Private Storage)
- REST APIs
- Encryption & Token-based security
